# 搬运视频工具

视频去重处理软件，支持多种去重功能。

## 功能特性

- ✅ 水平镜像翻转
- ✅ RGB颜色偏移
- ✅ 时间跳跃（帧率微波动）
- ✅ 修改MD5值
- ✅ 蒙版倒置
- ✅ 视频抽针
- ✅ 添加噪点
- ✅ 色彩调整（亮度/对比度）
- ✅ 速度微调

## 预设模式

| 模式 | 功能 | 适用场景 |
|------|------|----------|
| 快速 | 仅修改MD5 | 批量处理，平台检测不严格 |
| 标准 | MD5 + 时间跳跃 + 轻微噪点 | 日常使用推荐 |
| 强力 | MD5 + 时间跳跃 + RGB偏移 + 噪点 + 色彩微调 | 平台检测严格时使用 |
| 自定义 | 手动勾选功能 | 有特殊需求 |

## 使用方法

1. 下载项目
2. 运行 `搬运视频工具.bat`
3. 选择视频文件
4. 选择去重模式
5. 点击"开始处理"

## 环境要求

- Python 3.8+
- FFmpeg（需配置路径）
