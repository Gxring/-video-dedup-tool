视频去重处理软件，支持多种去重功能。

## 功能介绍

| 功能 | 说明 |
|------|------|
| 🪞 水平镜像 | 画面左右翻转，视觉效果焕然一新 |
| 🎨 RGB偏移 | 颜色通道错位，打造独特色调 |
| ⏱️ 时间跳跃 | 帧率微妙波动，肉眼难以察觉 |
| 🔐 修改MD5 | 重新编码+元数据，彻底改变文件指纹 |
| 🎭 蒙版倒置 | 调整透明度，增加画面层次感 |
| ✂️ 视频抽针 | 抽取特定帧，精简视频内容 |
| 📺 添加噪点 | 随机噪点干扰，提升去重效果 |
| 🌈 色彩调整 | 亮度/对比度微调，画面更自然 |
| ⚡ 速度微调 | 播放速度细微变化，不易被识别 |

## 预设模式

| 模式 | 策略 | 推荐场景 |
|------|------|----------|
| 🚀 快速 | 仅修改MD5 | 批量搬运，平台检测宽松 |
| ⭐ 标准 | MD5 + 时间跳跃 + 轻噪点 | 日常使用，效果与速度兼顾 |
| 💪 强力 | 全部去重手段叠加 | 平台检测严格，追求高通过率 |
| 🎛️ 自定义 | 自由组合功能 | 有特殊需求的老手 |

## 使用教程

1. 下载本项目
2. 双击运行 `搬运视频工具.bat`
3. 点击"浏览"选择视频文件
4. 选择合适的去重模式
5. 点击"开始处理"，等待完成

## 运行环境

- Python 3.8+
- FFmpeg（需配置到系统路径）
